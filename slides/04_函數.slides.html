Python函數類型¶

  • 在Python中有兩種不同的函數
    • 內建函數 (Built-in Functions)：Python預設提供
      • e.g. print(), type(), float(), int()
    • 自訂函數 (Function that we define ourselves)：我們自己寫的並且之後拿來使用
    • 我們把內建函數視為保留字，也就是其他變數的命名不能夠使用

定義函數¶

  • 在Python中，函數是一些可重用的程式碼，可以定義輸入的引數(arguments)，在程式碼中進行一些計算，然後從函數中回傳一個或多個結果
  • 定義函數時，使用保留字def
  • 使用函數，成為呼叫(call)或是調用(invoke)
  • 呼叫函數是使用函數的名稱，並加上括號結尾，在括號中可寫下要傳入到函數中的引數(argument)

建立自訂函數 Building our Own Functions¶

呼叫定義好的函數¶

  • 當我們定義好一個函數後，我們可以呼叫(call)或調用(invoke)函數，想呼叫幾次就幾次
  • 這就是儲存並重複使用的模式 (store and reuse pattern)
In [8]:
x = 5
print('Hello')

# 定義函數
def print_lyrics():
    print("I am a lumberjack, and I'm okay.")
    print("I sleep all night and I work all day.")

print('Yo')

# 呼叫函數
print_lyrics()

x = x + 2
print(x)

引數 Arguments¶

  • 引數是我們在呼叫(call)/調用(invoke)函數時，傳遞給函數當作輸入的數值
  • 我們可以在每次使用函數時，傳入不同的引數，讓函數去執行不同的工作
  • 我們將引數放在函數名稱後的括號中 
    big = max('Hello World')
    • 上面的程式碼，'Hello World'就是引數Argument

參數 Parameters¶

  • 參數是我們在函數定義中使用的變數，它允許函數中的程式碼訪問傳遞到函數的引數(argument)
In [9]:
def greet(lang):
    # lang是參數parameter
    if lang == 'es':
        print('Hola')
    elif lang == 'fr':
        print('Bonjour')
    else:
        print('Hello')

# 'en'是引數 argument
greet('en')

# 'es'是引數 argument
greet('es')

# 'fr'是引數 argument
greet('fr')

Hola
Bonjour

回傳值1 Return Value¶

  • 函數通常會接受其參數，進行一些計算並返回一個值
  • 這個值就是函數運算的結果，回到原本呼叫(call)/調用(invoke)函數的程式碼中
  • return關鍵字就是用於此目的
In [10]:
def greet():
    return 'Hello'

print(greet(), 'Glenn')
print(greet(), 'Sally')

Hello Glenn
Hello Sally

回傳值2 Return Value¶

  • 一個有「產出(fruitfull)」的函數是會產生結果(或返回值)的函數
  • return便結束了函數的執行，並回傳函數運作的結果
In [11]:
def greet(lang):
    if lang == 'es':
        return 'Hola'
    elif lang == 'fr':
        return 'Bonjour'
    else:
        return 'Hello'

print(greet('en'), 'Glenn')
print(greet('es'), 'Sally')
print(greet('fr'), 'Michael')

Hello Glenn
Hola Sally
Bonjour Michael

多引數/參數 Multiple Arguments/Parameters¶

  • 我們可以在函數定義中定義多個參數
  • 呼叫/調用函數時，我們只需添加更多引數
  • 我們要讓呼叫函數時，傳遞的引數與函數的參數數量與順序相同
In [12]:
# 定義了兩個參數
def addtwo(a, b):
    added = a + b
    return added

# 呼叫/調用函數時，引數的順序和數量要與函數的參數相同
x = addtwo(3, 5)
print(x)

無產出函數 Void Function¶

  • 當一個函數沒有回傳值時，我們稱為Void Function
  • 有回傳值的函數稱為有產出(fruitful)函數
  • 沒有回傳值的函數是無產出(not fruitful)函數

要轉成函數/不要轉成函數¶

  • 將您的程式碼組織為一段一段(paragraphs)的，以便釐清完整想法並為其命名
  • 別重複做同樣的事情，將程式碼處理好並能正常執行後重複使用它
  • 如果有些程式碼的邏輯太長或太複雜，將程式碼的邏輯切分成較小的區段，每個區段獨立成為一個函數
  • 可以為常常重複使用的程式碼建立資料庫，也許還可以與您的朋友分享

作業¶

  • 寫一個程式，跳出詢問框，詢問用戶的工作時數(hours)以及時薪(rate)並計算薪資總額
  • 40小時以下為正常工資，40小時以上，每小時為1.5倍工資

  • 函數應該要回傳計算的結果
  • 測試時使用工作時數45小時，時薪10.50來計算，所得結果應該為498.75
    • 提醒1：記得將用戶輸入的內容由字串轉換成浮點數
    • 提醒2：我們先假設用戶都會輸入數字而不是非數字的內容
    • 提醒3：變數的命名不要用sum，因為sum()是內建函數
