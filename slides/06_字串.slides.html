Chapter 06 字串(Strings)¶

字串的資料型態¶

  • 字串是一連串的字(character)
  • 字串需要用引號包住，像是"Hello"或'Hello'
In [ ]:
str1 = 'Hello'
str2 = "there"
  • 對於字串來說，兩個字串用+號連接，表示連接兩個字串的內容
In [ ]:
bob = str1 + str2
print(bob)

Hellothere
  • 當字串就算是數字，它仍然是個字串
In [ ]:
str3 = '123'
str3 = str3 + 1

  • 索引值一定要是從0開始的整數
  • 索引值可以是一個計算得出結果的表達式

字串有長度¶

  • 使用內建函數len()可以取得字串的長度
In [ ]:
fruit = 'banana'
print(len(fruit))

6

使用迴圈尋訪字串(1)¶

  • 使用while以及迭代變數再加上len()函數，我們可以建立一個可以尋訪字串中每一個字的迴圈
In [ ]:
fruit = 'banana'
index = 0
while index < len(fruit):
    letter = fruit[index]
    print(index, letter)
    index = index + 1

1 a
2 n
3 a

使用迴圈尋訪字串(2)¶

  • 使用for迴圈，表達方式會更簡潔，因為迭代變數完全由for迴圈來控制
In [ ]:
fruit = 'banana'
index = 0
for letter in fruit:
    print(index, letter)
    index = index + 1

迴圈與計數¶

  • 這是一個簡單的回圈，會去尋訪字串中每一個字並且計算a出現的次數
In [ ]:
word = 'banana'
count = 0
for letter in word:
    if letter == 'a':
        count = count + 1
print(count)

切割字串 (Slicing String)¶

  • 我們可以使用冒號運算子查看字串的任何連續部分
  • 冒號運算子的第二個數字是字串的結束位置，但不包含該位置
  • 如果冒號運算子的第二個數字大於字串結束的位置，則最長就只到字串的結尾
In [1]:
s = 'Monty Python'
print(s[0:4])
print(s[6:7])
print(s[6:20])

Mont
P
Python
  • 如果讓冒號運算子的第一個數字或第二個數字為空(不寫任何數字)，第一個數字不寫就預設為字串的開頭、第二個數字不寫就預設為字串的結尾
In [2]:
s = 'Monty Python'
print(s[:2])
print(s[8:])
print(s[:])

thon

使用in當作字串的邏輯運算子¶

  • in這個字除了使用在for迴圈，也可以使用在檢查某一段字串是不是「有包含(in)」特定的字串在其中
  • 所以in的檢查會表示True或False，所以可以用在if的判斷式中
In [6]:
fruit = 'banana'
print('n' in fruit)
print('m' in fruit)
print('nan' in fruit)
if 'a' in fruit:
    print('Found it!')

找尋字串 find()¶

  • 我們使用find()來尋找某些特定的文字/字串在其他字串中的位置
  • find()會回傳字串中首先找到的位置
  • 如果找不到，find()會回傳-1
  • 要記得，字串開始的位置是0
In [2]:
fruit = 'banana'
pos = fruit.find('na')
print(pos)

aa = fruit.find('z')
print(aa)

2
-1

所有字轉為大小寫 upper()/lower()¶

  • 你可以讓字串轉為完全大寫或小寫
  • 我們多半會把文字轉為小寫在進行find()，這樣不管文字大小寫都可以正確找到
In [ ]:
greet = 'Hello Bob'
nnn = greet.upper()
print(nnn)

www = greet.lower()
print(www)

替換文字 replace()¶

  • replace()可以尋找特定文字並替換掉
  • replace()會將所有找到的特定文字全部都替換掉
In [3]:
greet = 'Hello Bob'
nstr = greet.replace('Bob', 'Jane')
print(nstr)

nstr = greet.replace('o', 'X')
print(nstr)

Hello Jane
HellX BXb

去除空白 strip()¶

  • 我們可以使用以下方法去除字串開頭結尾的空白
  • lstrip()移除字串左方的空白
  • rstrip()移除字串右方的空白
  • strip()移除左方與右方的空白
In [4]:
greet = '   Hello Bob   '
print('---' + greet.lstrip() + '---')

print('---' + greet.rstrip() + '---')

print('---' + greet.strip() + '---')

---Hello Bob   ---
---   Hello Bob---
---Hello Bob---
